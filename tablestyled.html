The table below provides an overview of 40 HTML tags,
attributes and functions commonly used in web development

Tag Name Attribute functions
html - Root element of an HTML document
head - contains meta-information
tittle - sets the document tittle
meta - provides metadata
link - defines relationships with external resources
style - contains inline css styles
script - embeds or references javascript code
body - contains the visible content of the document
h1 to h6 - define headings of different levels
p - represents a paragraph
a - create hyperlinks
img - embeded images
ul - defines an unordered list
ol - defines an ordered list
li - represents a list item
div - a generic container for grouping elements
span - a generic inline container
form - defines an HTML form for user input
input - creates an input element
textarea - defines a multiline of text input
select - creates a dropdown menu
button - defines a clickable button
table - defines a table
tr - represents a table row
th - defines a table header cell
td - defines a table data cell
iframe - embeds an inline frame
audio - embeds audio contents
video - embeds video contents
canvas - used for drawing graphics with javascript
svg - embeds scalable vector graphics
footer - defines the footer of a document
nav - represents a navigation menu
aside - defines content aside from the main content
section - represents a thematicsection of content
header - defines the header of a document
strong - indicates strong importance
em - emphasizes text (slanty text)
blockquote - defines a block of quoted text
hr - creates a horizontal rule
src - specifies source URL for elements like img and script
href - defines the URL for hyperlinks created with (a)
alt - provides alternative text for images
id - gives a unique identifier to an element
class - assigns a class to an element for styling and scripting
onclick - - executes javascript when an element is clicked
onsubmit - executes javascript when a form is submitted
